This project analyses bus service coverage and accessibility across the Greater Melbourne area for Public Transport Victoria (PTV) using spatial and transport data.

Following the return to face-to-face work after COVID-19 restrictions, demand for public transport has increased. This analysis evaluates how well current bus services meet community needs, identifies underserved areas, and highlights regions with strong transport access.

Key Features

Uses PTV GTFS data (17 March 2023) to analyse bus routes, stops, and services.

Integrates ABS ASGS spatial boundaries (Mesh Blocks and allocation files) for geographic analysis.

Performs SQL-based data analytics and spatial aggregation by suburb, municipality, or postcode.

Produces visual insights to support transport planning and decision-making.

Objectives

Measure current bus accessibility across Greater Melbourne.

Identify gaps in public transport coverage.

Compare accessibility between different regions.

Population Reference

Greater Melbourne population (ABS Census 2021)
